# Cisco Packet Tracer Networking Labs

This repository contains hands-on networking labs built using **Cisco Packet Tracer** as part of my preparation for an entry-level **Network Technician** role.

## Skills Demonstrated
- IPv4 addressing and subnetting
- VLAN configuration and trunking
- Inter-VLAN routing (Router-on-a-Stick)
- Static routing
- DHCP, DNS, and NAT
- Access Control Lists (ACLs)
- Network troubleshooting using the OSI model

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI
- GitHub for documentation and version control

## Labs Included
- VLAN and Inter-VLAN Routing
- DHCP and NAT Configuration
- Network Security with ACLs
- Enterprise-Style Capstone Network

Each lab includes:
- Packet Tracer `.pkt` file
- Network topology diagram
- Configuration explanation
- Troubleshooting notes and validation steps
